## Lista de Exercícios – Algoritmos (Objetos, Arrays e Funções)

### 1. Cadastro de Aluno
Crie um objeto que represente um aluno com nome, idade e curso. Crie uma função que receba esse objeto e retorne uma frase apresentando o aluno.

### 2. Lista de Produtos
Crie um array contendo objetos de produtos com nome e preço. Crie uma função que receba o array e retorne apenas os produtos com preço acima de 50.

### 3. Média de Notas
Crie uma função que receba um array de números representando notas e retorne a média final.

### 4. Busca de Nome
Crie um array com nomes. Crie uma função que receba um nome e retorne se ele existe no array.

### 5. Contar Ocorrências
Crie uma função que receba um array e retorne um objeto mostrando quantas vezes cada elemento aparece.

### 6. Biblioteca
Crie um array de objetos representando livros com título, autor e ano. Crie uma função que retorne apenas os livros publicados depois de 2010.

### 7. Carrinho de Compras
Crie um array com objetos de itens (nome, quantidade, preço). Crie uma função que calcule o valor total do carrinho.

### 8. Filtrar Pessoas
Crie um array de objetos contendo nome e idade. Crie uma função que retorne apenas as pessoas com mais de 18 anos.

### 9. Sistema de Caixa Eletrônico
Crie um objeto representando uma conta com agência, número da conta, senha, saldo e histórico de operações.
Crie um sistema com as funções:
- validar acesso (recebe agência, conta e senha)
- exibir saldo
- realizar saque (somente se houver saldo)
- registrar no histórico cada operação
No saque, exiba a quantidade de cédulas entregues usando as notas: 100, 50, 20 e 10, utilizando o menor número possível de cédulas.

### 10. Ranking de Pontuação
Crie um array de objetos representando jogadores com nome e pontos. Crie uma função que retorne o array ordenado do maior para o menor ponto.
